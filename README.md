Jmiac Clothing Website

This is a multi-page clothing website built using HTML and CSS.
The goal of the project was to create a clean, responsive, and user-friendly interface for browsing clothing items.

Features
       Multi-page structure (Home, About, Products, Contact)
       Responsive layout that works on mobile and desktop
       Product section with images and descriptions
       Navigation between pages
       Basic styling using CSS and Flexbox
       
Technologies Used

HTML5
CSS3 (including Flexbox)
